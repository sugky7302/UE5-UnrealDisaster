## 安裝

1. 下載此專案。
2. 利用 VS Code 加載 .devcontainer，創建 Docker 環境。
3. 安裝 `pnpm`。

```
npm i -g pnpm
```

4. 安裝依賴套件。

```
cd /workspaces/tsc-rpg-practice
pnpm i
```
